# 나만의 날씨 웹사이트

## 프로젝트 개요
동적 날씨 웹사이트 제작을 목표로 한 프로젝트입니다.  
OpenWeatherMap API를 활용하여 현재 날씨, 3~5일 단기 예보, 시간별 그래프, 공기질 정보를 표시합니다.  
Vercel을 통해 배포하며, 환경변수를 활용해 API Key를 안전하게 관리합니다.

---

## 주요 기능

### 필수 기능
- **도시 검색:** 입력 후 버튼 클릭 또는 Enter 이벤트로 검색 가능
- **API 연동:** OpenWeatherMap API 호출, JSON 데이터 파싱
- **날씨 표시:** 현재 온도, 상태, 아이콘, 습도, 풍속
- **예보 카드:** 3~5일 단기 예보 생성, 날짜/온도/상태 표시
- **단위 전환:** 섭씨(°C) ↔ 화씨(°F) 전환 버튼
- **시각적 변화:** 날씨 및 단위 변경 시 동적 표시
- **오류 처리:** 잘못된 도시 입력 및 API 호출 실패 시 알림
- **최근 검색어 저장:** 최대 5개, localStorage 활용

### 선택 확장 기능
- **위치 자동 감지 (Geolocation API)**
- **미세먼지/공기질 표시**
- **시간별 그래프/차트 표시 (Chart.js 활용)**

---

## 기술 스택
- **HTML / CSS / JavaScript (Vanilla JS)**
- **Chart.js:** 시간별 온도 그래프
- **OpenWeatherMap API:** 날씨, 예보, 공기질
- **Vercel:** 배포 및 환경변수 관리
